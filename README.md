
# Установка

когда скачали папку ,то переходим в каталог папки

```shell
cd Ready_ant_server
```
и после вводим 
```shell
npm install 
```
чтобы скачать все зависимости

## Запуск
переходим в каталог 

```shell
cd Ready_ant_server
```
и после вводим команду
```shell
node ./server.js
```
Следующий шаг - переходим в браузер и вбиваем "http://localhost:5000/"
и се ,чудо запустилось
